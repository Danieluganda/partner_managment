## Troubleshooting

### Dashboard shows "No data"
- Make sure you're using `dashboard_standalone.html`, OR
- Make sure all 4 files are in the same folder

### Tables are empty
- Press F12 to open browser console
- Look for error messages
- Try refreshing the page (Ctrl+R or Cmd+R)

### Nothing happens when I click
- Make sure JavaScript is enabled in your browser
- Try a different browser (Chrome, Firefox, Edge)

## Updating Your Data

To update with new Excel data:

1. Replace your Excel files
2. Re-run the Python conversion script:
```python
python3 convert_data.py
```
3. Refresh your browser

Or manually edit `dashboard_data.json` in any text editor.
